Гость - из центра когнитивных исследований

Для обзоров сущ-ют конкретные протоколы - строгий подход

Шаг 1 - составление грамотного запроса
	- можно применять поиск по ключевым словам с применением операторов
	например: ( kw1 & kw2 ) or kw3 и т.п.

Шаг 2 - Выгрузка данных
	RIS - формат - это такой формат который содержит мета данные по статьям: 
		название, авторы, абстракт, год и журнал

Шаг 3 - Скрининг абстрактов
	Сервисы для оценки абстрактов: Abstrackr, Zotero, 
	<b>Nested knowledge</b> (single project in a time), Mendeley

Шаг 4 - Полнотекстовый скрининг
	ResearchGate - источник статей от авторов - кидаем запрос и получаем если коннект

Шаг 5 - Экстракция данных
	Экстракция данных - детальная разметка данных статьи по категориям / конспектирование

Шаг 6 - Написание текста
